Monomer Structural Features

Structural properties at the monomer level are described in the categories STRUCT_MON_DETAILS, STRUCT_MON_PROT, STRUCT_MON_NUCL, and STRUCT_MON_PROT_CIS. These categories contain features such as: backbone torsion angles, isotropic temperature factors, real space R factors, and real space correlation coefficients. Precise details about the calculation of the latter quantities may be specified in the STRUCT_MON_DETAILS category. The structural features described in these categories can be identified using standard mmCIF monomer labels and/or an alternative nomenclature preferred by the author.
