Introduction to the Contents of the mmCIF Dictionary

This tutorial provides a tour of the some of the category groups in the mmCIF dictionary which describe macromolecular structure. Some of the dictionary terminology used to describe elements of molecular structure are also described. Each group of categories is presented in the form of a schematic diagram and a set of abbreviated examples. Category diagrams are provided for each example.

The category diagrams are organized in the following manner:

  • Boxes enclose the the data items within each mmCIF category.
  • Key data items are preceded by black dots.
  • Data items common to multiple categories are identified by connecting lines with the arrows pointing at the parent definition of the data item.
